# Billing

## Summary

- OpenMates uses credits for cloud-hosted paid usage.
- Users can review usage, understand pricing, redeem gift cards, and export account data from the existing billing-related guides.
- Self-hosted installs do not use cloud payment flows; see the self-hosting guide for self-host-specific setup.

## Billing Guides

- [Pricing](pricing.md) -- understand credits and what gets charged.
- [Usage & Billing](usage.md) -- view credit usage and export usage history.
- [Gift Cards](gift-cards.md) -- redeem gift card codes for credits.
- [Invoices and Refunds](invoices-and-refunds.md) -- invoice history and refund boundaries.
- [Self-Hosting](../../self-hosting/README.md) -- install without cloud billing or payment flows.
