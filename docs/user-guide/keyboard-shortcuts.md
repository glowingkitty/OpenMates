# Keyboard Shortcuts

> Speed up your workflow with keyboard shortcuts. All shortcuts adapt automatically to your platform -- use `Cmd` on macOS and `Ctrl` on Windows/Linux.

## Quick Reference

| Action | Windows / Linux | macOS | When |
|--------|----------------|-------|------|
| **New chat** | `Ctrl + K`, then `N` | `Cmd + K`, then `N` | Anywhere |
| **Toggle chat list** | `Ctrl + Shift + H` | `Cmd + Shift + H` | Anywhere |
| **Next chat** | `Ctrl + Shift + Right` | `Cmd + Shift + Right` | Anywhere |
| **Previous chat** | `Ctrl + Shift + Left` | `Cmd + Shift + Left` | Anywhere |
| **Download chat** | `Ctrl + Shift + S` | `Cmd + Shift + S` | Chat open |
| **Copy chat** | `Ctrl + Shift + C` | `Cmd + Shift + C` | Chat open |
| **Focus message input** | `Shift + Enter` | `Shift + Enter` | Input not focused |
| **Scroll to top** | `Ctrl + Shift + Up` | `Cmd + Shift + Up` | In chat |
| **Scroll to bottom** | `Ctrl + Shift + Down` | `Cmd + Shift + Down` | In chat |

## How It Works

### Command Palette (New Chat)

The new chat shortcut uses a two-step command palette pattern:

1. Press `Ctrl/Cmd + K` to enter command mode.
2. Press `N` within 2 seconds to create a new chat.
3. Press `Esc` to cancel.

This two-step approach avoids conflicts with browser shortcuts and makes it easy to add more commands in the future.

### Chat Navigation

- **Next / Previous chat** moves through your chat list, loading each chat as you go.
- Works even when the chat list sidebar is closed.

### Scroll Shortcuts

- **Scroll to top** jumps to the oldest messages in the conversation.
- **Scroll to bottom** jumps to the newest messages.

### Focus Message Input

Press `Shift + Enter` when the message input is not focused to place your cursor there. This does not send a message -- it just moves focus.

## Browser Notes

Most shortcuts work reliably across Chrome, Firefox, Safari, and Edge. A few things to be aware of:

- `Ctrl/Cmd + Shift + N` may still open a private/incognito browser window in some browsers. Use the on-screen button as an alternative.
- On macOS, some shortcuts may be captured by the operating system before they reach the web app.

All keyboard actions also have on-screen equivalents (buttons, menus), so you can always use those instead.

## Tips

- Every shortcut has a matching button or menu option in the interface -- keyboard shortcuts are optional.
- If a shortcut conflicts with assistive technology you use, you can rely on the on-screen controls instead.
