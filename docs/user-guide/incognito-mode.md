# Incognito Mode

> Create temporary chats that exist only in your current browser tab. When you close the tab or turn off incognito mode, they are gone.

## What It Does

Incognito mode lets you have conversations that are never stored on the server or synced to other devices. Incognito chats live only in your current browser tab's memory. They are ideal for one-off questions or sensitive topics you do not want saved anywhere.

## How to Use It

### Turning It On

1. Open **Settings** (quick settings section).
2. Toggle **Incognito mode** on.
3. The first time you enable it, an explainer will appear describing how it works.

### While Active

- **New chats** you create are incognito chats. They appear in your chat list with a darker background and an "Incognito" label.
- **Existing chats** are not affected. Sending a message in an existing regular chat does not make it incognito.
- Incognito chats survive a page refresh (within the same tab) but are gone when you close the tab.

### Turning It Off

When you disable incognito mode:

- All incognito chats are **immediately and permanently deleted** from memory.
- If you have an incognito chat open, it is closed automatically.
- New chats you create afterwards will be regular (stored and synced) chats.
- There is no way to recover deleted incognito chats.

## How It Differs from Regular Chats

| | Regular chat | Incognito chat |
|---|---|---|
| **Stored on server** | Yes (encrypted) | No |
| **Synced across devices** | Yes | No |
| **Survives tab close** | Yes | No |
| **Follow-up suggestions** | Yes | No |
| **Usage tracked** | Per chat | Grouped as "Incognito" |

## Multiple Tabs

Each browser tab has its own incognito mode state. Incognito chats in one tab are not visible in another tab, even on the same device.

## Billing

Incognito chats still use credits. Your usage is tracked, but all incognito activity is grouped under a single "Incognito chats" label in your usage history -- individual chat details are not recorded.

## Tips

- Incognito mode is per-tab, so you can have one tab in incognito mode and another in regular mode at the same time.
- If you want to keep a conversation that started in incognito mode, copy the text before turning off the mode or closing the tab.
