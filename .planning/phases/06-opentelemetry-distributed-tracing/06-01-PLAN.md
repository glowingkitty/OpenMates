    4. Create `backend/shared/python_utils/tracing/privacy_filter.py`:
       - IMPORTANT: OTel Python SDK's `on_end()` receives immutable `ReadableSpan`. Implement as a **wrapping SpanExporter** (not SpanProcessor) that filters attributes during `export()`.
       - Class `TracePrivacyFilter` implementing `SpanExporter`:
         - Constructor takes `inner: SpanExporter` (the real OTLP exporter)
         - `export(self, spans: Sequence[ReadableSpan]) -> SpanExportResult`:
           - For each span, determine tier via `determine_user_tier(span)`
           - Create a filtered copy of span attributes based on tier
           - Forward filtered spans to `inner.export()`
         - `shutdown()` and `force_flush()` delegate to `inner`
       - Tier attribute lists (define as module-level constants):
         - `TIER_3_ONLY_ATTRS`: attributes only kept at Tier 3: `{"ws.payload_size", "cache.key", "cache.value", "llm.timing", "llm.token_count", "skill.params"}`
         - `TIER_2_ONLY_ATTRS`: attributes only kept at Tier 2+: `{"exception.stacktrace", "http.request.header.authorization_type", "http.request.header.content_type", "cache.hit", "db.query_timing", "celery.task_id", "celery.queue"}`
         - `ALWAYS_STRIP_ATTRS`: attributes NEVER exported: `{"http.request.header.cookie", "http.request.header.authorization"}`
       - `_pseudonymize_user_id(user_id: str) -> str`:
         - SHA256 of `f"{user_id}:{daily_salt}"`, return first 12 hex chars
         - Daily salt: SHA256 of `f"otel-salt:{date.today().isoformat()}"`, first 16 hex chars
         - Cache the salt and date, regenerate when date changes
       - Dev server check: `os.getenv("SERVER_ENVIRONMENT", "dev") == "dev"` -> skip all filtering

    5. Create `backend/shared/python_utils/tracing/user_tier.py`:
       - `determine_user_tier(span_attributes: dict) -> int`:
         - If `span_attributes.get("enduser.is_admin")` is True -> return 3
         - If `span_attributes.get("enduser.debug_opted_in")` is True -> return 3
         - If span has error status (check `otel.status_code` == "ERROR") -> return 2
         - Else -> return 1
       - Note: The actual user lookup (is_admin, debug_opted_in) will be injected by WS handlers and middleware in Plan 03. For now, `determine_user_tier` reads from span attributes that callers set.
