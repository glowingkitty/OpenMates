    `start_ws_handler_span(handler_name: str, user_id: str, payload: Optional[dict], user_otel_attrs: Optional[dict] = None) -> Tuple[Any, Any]`:
    - ImportError-guarded (returns (None, None) if OTel not installed)
    - If payload is not None, calls extract_ws_trace_context(payload) for parent context
    - Creates span named f"ws.{handler_name}" with attributes:
      - "ws.message_type": handler_name
      - "enduser.id": str(user_id)
      - "enduser.is_admin": user_otel_attrs.get("is_admin", False) if user_otel_attrs else False

    - Attaches span to context, returns (span, token)

    `end_ws_handler_span(span, token, error: Optional[Exception] = None) -> None`:
    - If error and span: span.set_status(StatusCode.ERROR, str(error))
    - If span: span.end()
    - If token: context.detach(token)
    - All wrapped in try/except (never raises)
