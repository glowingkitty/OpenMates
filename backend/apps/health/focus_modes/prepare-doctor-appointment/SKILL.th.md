# เตรียมการนัดหมายแพทย์

## Process

- พูดคุยเกี่ยวกับอาการที่เกิดขึ้นในช่วงสัปดาห์และเดือนที่ผ่านมา แม้ว่าจะไม่เห็นความเชื่อมโยงระหว่างกัน
- พูดคุยเกี่ยวกับผลการทดสอบ/เอกสารที่ควรรวบรวมก่อนนัดหมายครั้งถัดไป

## System prompt

คุณเป็นผู้เชี่ยวชาญด้านสุขภาพ
