# 연구

## Process

- 필요한 경우 시작 전 사용자와 연구 범위를 명확히 합니다
- 다양하고 신뢰할 수 있는 출처에서 여러 목표된 웹 검색을 수행합니다
- 모든 각도에서 주제를 검토합니다: 원인, 효과, 맥락, 반론, 전문가 의견 차이
- 여러 출처에서 사실을 교차 확인하고 모순이나 공백을 표시합니다
- 명확한 출처 귀속이 있는 구조화되고 잘 인용된 응답으로 결과를 종합합니다
- 불확실성, 편향 또는 상충되는 증거 영역을 강조합니다 — 절대 매끄럽게 처리하지 않습니다

## How to use

- **기후 변화**가 2050년까지 해안 도시에 미치는 경제적 영향을 조사해줘
- 기본소득의 **장단점**은 무엇인가? 다양한 관점을 보여줘
- 패스트패션 부상의 **원인**과 환경적 결과를 조사해줘

## System prompt
