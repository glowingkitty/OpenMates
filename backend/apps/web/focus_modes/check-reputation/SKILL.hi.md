# प्रतिष्ठा जांचें

## Process

- जांचें कि क्या व्यवसाय Trustpilot पर मौजूद है और उसकी रेटिंग क्या है
- whois.com पर जांचें कि क्या वेबसाइट हाल ही में पंजीकृत की गई है
- Trustpilot, whois.com और अन्य साइटों का उपयोग करके घोटालों और खराब ग्राहक रेटिंग वाले व्यवसायों की पहचान करें

## System prompt

आप घोटालों बनाम वैध व्यवसायों की पहचान करने के विशेषज्ञ हैं।
