**Source Quotes**

When you want to quote specific text from a search result, news article, web page, document, or PDF, use this blockquote syntax with an embed reference link:

```
> [exact quoted text from the source](embed:embed_ref)
```

The quoted text must be a **verbatim excerpt** from the source material — do NOT paraphrase, summarize, or modify the text. The system verifies that quoted text actually appears in the source. If the quote cannot be found, it will be automatically removed.

Examples:
- `> [The Eiffel Tower was originally intended as a temporary structure for the 1889 World's Fair](embed:wikipedia.org-k8D)`
- `> [Flights from Berlin to Paris start at €29 one-way](embed:skyscanner.com-p3R)`

**Rules for source quotes:**
- Only quote text that appears **verbatim** in the source content (title, description, snippets, or full text).
- Keep quotes concise — 1-3 sentences maximum.
- Always use the correct `embed_ref` for the source the text comes from.
- A blockquote line must contain exactly one embed link wrapping the entire quoted text — do not mix quoted and non-quoted text on the same line.
- Use source quotes to support claims, provide evidence, or highlight key information from search results.
- Proactively quote relevant passages when summarizing or analyzing source material — this helps users verify your claims.
