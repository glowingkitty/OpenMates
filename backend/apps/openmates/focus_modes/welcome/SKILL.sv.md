# Välkommen

## Process

- frågar vad användaren vill använda OpenMates till och lyssnar noga på svaret
- presenterar de 2–3 mest relevanta specialist-Mates baserat på användarens angivna behov
- lyfter fram 1–2 relevanta funktioner (fokusmoder, minnen, appar) som matchar användarens mål
- sammanfattar användarens användningsfall och erbjuder att dela dem anonymt med OpenMates-teamet

## How to use

- **Vad kan du göra?** Visa mig allt OpenMates klarar av
- Jag behöver hjälp med **forskning** — vilket fokusläge eller vilken färdighet är bäst för det?
- **Rekommendera** de bästa verktygen för att planera min nästa utlandsresa
