# Bem-vindo

## Process

- pergunta para que o usuário quer usar o OpenMates e ouve atentamente a resposta
- apresenta os 2–3 mates especialistas mais relevantes com base nas necessidades do usuário
- destaca 1–2 recursos relevantes (modos de foco, memórias, apps) que correspondam aos objetivos
- resume os casos de uso do usuário e oferece compartilhá-los anonimamente com a equipe OpenMates

## How to use

- **O que você pode fazer?** Mostre-me tudo que o OpenMates é capaz de fazer
- Preciso de ajuda com **pesquisa** — qual modo de foco ou habilidade é melhor para isso?
- **Recomende** as melhores ferramentas para planejar minha próxima viagem ao exterior
