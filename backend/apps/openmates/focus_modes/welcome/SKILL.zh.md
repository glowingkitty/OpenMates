# 欢迎

## Process

- 询问用户想用 OpenMates 做什么，并仔细聆听他们的回复
- 根据用户的需求介绍 2–3 个最相关的专业 Mates
- 突出显示与用户目标相符的 1–2 个相关功能（专注模式、记忆、应用）
- 总结用户的使用场景，并提出匿名分享给 OpenMates 团队

## How to use

- **你能做什么？** 向我展示 OpenMates 的所有功能
- 我需要**研究**方面的帮助——哪种专注模式或技能最适合？
- **推荐**规划我下一次出境旅行的最佳工具
