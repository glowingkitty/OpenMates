# ยินดีต้อนรับ

## Process

- ถามว่าผู้ใช้ต้องการใช้ OpenMates เพื่ออะไรและฟังคำตอบอย่างตั้งใจ
- แนะนำ Mate ผู้เชี่ยวชาญ 2–3 คนที่เกี่ยวข้องมากที่สุดตามความต้องการของผู้ใช้
- เน้น 1–2 คุณสมบัติที่เกี่ยวข้อง (โหมดโฟกัส ความทรงจำ แอป) ที่ตรงกับเป้าหมายของผู้ใช้
- สรุปกรณีการใช้งานของผู้ใช้และเสนอที่จะแบ่งปันโดยไม่ระบุตัวตนกับทีม OpenMates

## How to use

- **คุณทำอะไรได้บ้าง?** แสดงทุกสิ่งที่ OpenMates ทำได้ให้ฉันดูหน่อย
- ฉันต้องการความช่วยเหลือเรื่อง**การวิจัย** — โหมดโฟกัสหรือทักษะไหนดีที่สุดสำหรับสิ่งนั้น?
- **แนะนำ**เครื่องมือที่ดีที่สุดสำหรับการวางแผนการเดินทางต่างประเทศครั้งต่อไปของฉัน
