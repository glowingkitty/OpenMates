# Добро пожаловать

## Process

- спрашивает, для чего пользователь хочет использовать OpenMates, и внимательно слушает ответ
- представляет 2–3 наиболее релевантных специализированных mate на основе потребностей пользователя
- выделяет 1–2 релевантных функции (режимы фокуса, воспоминания, приложения), соответствующие целям пользователя
- подводит итоги сценариев использования и предлагает поделиться ими анонимно с командой OpenMates

## How to use

- **Что ты умеешь?** Покажи мне всё, на что способен OpenMates
- Мне нужна помощь с **исследованиями** — какой фокус-режим или навык лучше всего подходит для этого?
- **Порекомендуй** лучшие инструменты для планирования моей следующей поездки за рубеж
