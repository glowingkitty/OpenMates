# Bienvenido

## Process

- pregunta para qué quiere usar OpenMates y escucha atentamente la respuesta
- presenta los 2–3 mates especialistas más relevantes según las necesidades del usuario
- destaca 1–2 características relevantes (modos de enfoque, memorias, aplicaciones) que coincidan con los objetivos
- resume los casos de uso del usuario y ofrece compartirlos de forma anónima con el equipo de OpenMates

## How to use

- **¿Qué puedes hacer?** Muéstrame todo lo que OpenMates puede hacer
- Necesito ayuda con la **investigación** — ¿qué modo de enfoque o habilidad es mejor para eso?
- **Recomienda** las mejores herramientas para planificar mi próximo viaje al extranjero
