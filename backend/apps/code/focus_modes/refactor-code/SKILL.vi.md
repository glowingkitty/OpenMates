# Tái cấu trúc mã

## Process

- mate đặt nhiều câu hỏi để hiểu rõ hơn về yêu cầu và những gì không hoạt động tốt trong mã hiện tại
- mate kiểm tra tổng quan cấu trúc mã và đặt thêm câu hỏi làm rõ nếu cần
- mate cố gắng tìm mã dư thừa, độ phức tạp không cần thiết, v.v. và đề xuất cải tiến trước khi triển khai
- mate triển khai các thay đổi từng bước, yêu cầu phản hồi và sau đó tiếp tục bước tiếp theo
