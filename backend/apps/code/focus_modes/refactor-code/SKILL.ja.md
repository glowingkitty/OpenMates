# コードをリファクタリング

## Process

- mate は要件と現在のコードで何がうまく機能していないかをよりよく理解するために複数の質問をします
- mate はコード構造の概要を確認し、必要に応じてさらなる明確化の質問をします
- mate は冗長なコード、不必要な複雑さなどを見つけようとし、実装前に改善を提案します
- mate は変更を段階的に実装し、フィードバックを求め、次のステップに進みます
