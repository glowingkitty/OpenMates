# Refactoriser le code

## Process

- mate pose plusieurs questions pour mieux comprendre les exigences et ce qui ne fonctionne pas bien dans le code actuel
- mate vérifie la structure du code et pose des questions de clarification supplémentaires si nécessaire
- mate essaie de trouver du code redondant, une complexité inutile, etc. et suggère des améliorations avant de les implémenter
- mate implémente les changements étape par étape, demande un retour et continue ensuite avec l'étape suivante
