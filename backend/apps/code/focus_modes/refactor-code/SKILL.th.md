# รีแฟกเตอร์โค้ด

## Process

- mate ถามคำถามหลายข้อเพื่อทำความเข้าใจข้อกำหนดและสิ่งที่ไม่ทำงานดีในโค้ดปัจจุบัน
- mate ตรวจสอบภาพรวมโครงสร้างโค้ดและถามคำถามเพื่อความชัดเจนเพิ่มเติมหากจำเป็น
- mate พยายามหาโค้ดที่ซ้ำซ้อน ความซับซ้อนที่ไม่จำเป็น ฯลฯ และแนะนำการปรับปรุงก่อนนำไปใช้
- mate ดำเนินการเปลี่ยนแปลงทีละขั้น ขอข้อเสนอแนะ และดำเนินการต่อในขั้นตอนถัดไป
