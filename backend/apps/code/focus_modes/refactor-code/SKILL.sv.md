# Refaktorera kod

## Process

- mate ställer flera frågor för att bättre förstå kraven och vad som inte fungerar bra i den nuvarande koden
- mate kontrollerar kodstrukturens översikt och ställer ytterligare klargörande frågor vid behov
- mate försöker hitta redundant kod, onödig komplexitet etc. och föreslår förbättringar innan de implementeras
- mate implementerar ändringarna steg för steg, ber om feedback och fortsätter sedan med nästa steg
