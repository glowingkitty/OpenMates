# 重构代码

## Process

- mate 提出多个问题，更好地了解需求以及当前代码中哪些地方运行不好
- mate 检查代码结构概览，如有需要提出进一步的澄清问题
- mate 尝试找出冗余代码、不必要的复杂性等，并在实施之前建议改进措施
- mate 逐步实施变更，征求反馈，然后继续下一步
