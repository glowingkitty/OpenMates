# Refactoring codice

## Process

- mate fa più domande per capire meglio i requisiti e cosa non funziona bene nel codice attuale
- mate controlla la struttura del codice e fa ulteriori domande di chiarimento se necessario
- mate cerca codice ridondante, complessità non necessaria, ecc. e suggerisce miglioramenti prima di implementarli
- mate implementa le modifiche passo dopo passo, chiede feedback e poi continua con il passo successivo
