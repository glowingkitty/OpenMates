# कोड को रिफैक्टर करें

## Process

- mate आवश्यकताओं और वर्तमान कोड में क्या ठीक से काम नहीं कर रहा है, यह बेहतर ढंग से समझने के लिए कई प्रश्न पूछता है
- mate कोड संरचना का अवलोकन करता है और यदि आवश्यक हो तो अतिरिक्त स्पष्टीकरण प्रश्न पूछता है
- mate अनावश्यक कोड, अनावश्यक जटिलता आदि खोजने की कोशिश करता है और उन्हें लागू करने से पहले सुधार सुझाता है
- mate परिवर्तनों को चरण-दर-चरण लागू करता है, फीडबैक मांगता है और फिर अगले चरण पर जाता है
