# Рефакторинг кода

## Process

- mate задаёт несколько вопросов, чтобы лучше понять требования и что не работает хорошо в текущем коде
- mate проверяет обзор структуры кода и задаёт дополнительные уточняющие вопросы при необходимости
- mate пытается найти избыточный код, ненужную сложность и т.д. и предлагает улучшения перед их реализацией
- mate реализует изменения шаг за шагом, запрашивает обратную связь и затем переходит к следующему шагу
