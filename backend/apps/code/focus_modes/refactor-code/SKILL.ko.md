# 코드 리팩토링

## Process

- mate는 요구사항과 현재 코드에서 잘 작동하지 않는 것을 더 잘 이해하기 위해 여러 질문을 합니다
- mate는 코드 구조 개요를 확인하고 필요하면 추가 명확화 질문을 합니다
- mate는 중복 코드, 불필요한 복잡성 등을 찾고 구현 전에 개선사항을 제안합니다
- mate는 변경 사항을 단계별로 구현하고 피드백을 요청한 후 다음 단계로 진행합니다
