# Refaktoryzuj kod

## Process

- mate zadaje wiele pytań, aby lepiej zrozumieć wymagania i co nie działa dobrze w obecnym kodzie
- mate sprawdza przegląd struktury kodu i zadaje dalsze pytania wyjaśniające w razie potrzeby
- mate stara się znaleźć redundantny kod, niepotrzebną złożoność itp. i sugeruje ulepszenia przed ich wdrożeniem
- mate wdraża zmiany krok po kroku, prosi o informacje zwrotne, a następnie przechodzi do następnego kroku
