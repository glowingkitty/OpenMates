# Code refactoren

## Process

- mate stelt meerdere vragen om de vereisten en wat niet goed werkt in de huidige code beter te begrijpen
- mate controleert het overzicht van de codestructuur en stelt verdere verhelderende vragen indien nodig
- mate probeert redundante code, onnodige complexiteit enz. te vinden en stelt verbeteringen voor voordat ze worden geïmplementeerd
- mate implementeert de wijzigingen stap voor stap, vraagt om feedback en gaat dan verder met de volgende stap
