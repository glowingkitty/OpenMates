# Refaktorovat kód

## Process

- mate klade několik otázek, aby lépe porozuměl požadavkům a co nefunguje dobře v aktuálním kódu
- mate kontroluje přehled struktury kódu a klade další objasňující otázky dle potřeby
- mate se snaží najít redundantní kód, zbytečnou složitost atd. a navrhuje vylepšení před jejich implementací
- mate implementuje změny krok za krokem, žádá o zpětnou vazbu a poté pokračuje dalším krokem
