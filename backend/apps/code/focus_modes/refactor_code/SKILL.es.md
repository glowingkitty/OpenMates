# Refactorizar código

## Process

- mate hace múltiples preguntas para entender mejor los requisitos y qué no funciona bien en el código actual
- mate revisa la estructura del código y hace preguntas de aclaración adicionales si es necesario
- mate intenta encontrar código redundante, complejidad innecesaria, etc. y sugiere mejoras antes de implementarlas
- mate implementa los cambios paso a paso, pide retroalimentación y luego continúa con el siguiente paso
