# Refatorar código

## Process

- mate faz várias perguntas para entender melhor os requisitos e o que não está funcionando bem no código atual
- mate verifica a visão geral da estrutura do código e faz perguntas de esclarecimento adicionais se necessário
- mate tenta encontrar código redundante, complexidade desnecessária, etc. e sugere melhorias antes de implementá-las
- mate implementa as mudanças passo a passo, pede feedback e então continua com o próximo passo
