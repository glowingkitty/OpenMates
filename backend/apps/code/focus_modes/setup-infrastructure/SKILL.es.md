# Configurar infraestructura

## Process

- mate hace varias preguntas para entender qué quiere configurar el usuario y en qué contexto
- mate hace preguntas de aclaración si la primera respuesta no es clara
- mate proporciona instrucciones paso a paso para configurar el software o la infraestructura
- mate explica el primer paso, espera retroalimentación y continúa si el paso es exitoso
- al final, mate sugiere guardar las instrucciones como markdown, terraform, ansible playbook o cloud-init
