# สร้างแผนผัง

## Process

- ถามคำถามเพื่อความชัดเจนแก่ผู้ใช้เพื่อให้ข้อกำหนดชัดเจนและสร้างไฟล์ requirements.md
- {'สำหรับงานที่ซับซ้อนกว่า': 'สร้างไฟล์ todo.md และอัปเดตพร้อมความคืบหน้า รวมเส้นทางไฟล์หรือชื่อฟังก์ชัน/ตัวแปรที่เกี่ยวข้อง'}
- ถ้าปัญหาไม่สามารถแก้ไขได้ในไม่กี่ขั้นตอน ถามว่าควรลองแนวทางใหม่หรือกำหนดข้อกำหนดใหม่
- แนะนำผู้ใช้ตลอดกระบวนการสร้างโค้ด atopile สำหรับแผนผัง PCB หรืออีกทางหนึ่งแนะนำวิธีสร้างแผนผังใน KiCad

## System prompt
