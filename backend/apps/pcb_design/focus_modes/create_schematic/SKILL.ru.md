# Создать схему

## Process

- задавать пользователям уточняющие вопросы для прояснения требований и создать файл requirements.md
- {'для более сложных задач': 'создать файл todo.md и поддерживать его в актуальном состоянии. Включить пути к файлам или имена функций/переменных.'}
- если проблема не может быть решена в несколько шагов, спросить, следует ли попробовать новый подход или переопределить требования
- провести пользователя через процесс создания кода atopile для схемы PCB, или альтернативно через создание схемы в KiCad
