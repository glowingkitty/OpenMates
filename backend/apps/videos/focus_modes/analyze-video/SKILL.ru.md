# Анализировать видео

## Process

- получает полную транскрипцию видео YouTube с помощью навыка транскрипции
- суммирует основные моменты, аргументы и выводы видео
- определяет точку зрения создателя, потенциальные предубеждения и используемые риторические приёмы
- извлекает ключевые фактические утверждения из видео
- выполняет веб-поиски для проверки или оспаривания наиболее значимых утверждений
- синтезирует результаты в структурированный анализ с резюме, оценкой предвзятости, результатами проверки фактов и общей профессиональной оценкой
