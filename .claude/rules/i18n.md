## Critical: Never Edit Generated JSON

The `.json` files in `frontend/packages/ui/src/i18n/locales/` are **auto-generated**.

- Only edit `.yml` source files in `frontend/packages/ui/src/i18n/sources/`
- After editing `.yml` files, run: `cd frontend/packages/ui && npm run build:translations`
- This applies to ALL operations: adding keys, fixing typos, updating translations
- The auto-rebuild-translations hook will handle this automatically after `.yml` edits

For full translation management guide: `python3 scripts/sessions.py context --doc manage-translations`
